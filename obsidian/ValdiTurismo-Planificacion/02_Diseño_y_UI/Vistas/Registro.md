### Vista: `/registro` (Crear Cuenta)

**1. Contenedor Principal (La Isla Central)**

- **Encabezado:** Título invitando a unirse a la plataforma.
    
- **Sección de Registro Rápido (Social):**
    
    - Botón destacado para registrarse directamente con **Google** (ubicado arriba para reducir fricción).
        
- **Divisor Visual:** * Línea o espacio con el texto _"o regístrate con tu correo"_.
    
- **Sección de Datos Personales:**
    
    - Fila agrupada: Campo de entrada para **Nombre** y Campo de entrada para **Apellido**.
        
- **Sección de Demografía:**
    
    - Selector o lista desplegable: **¿De dónde vienes?** (Opciones predefinidas de ciudades/países para evitar texto libre).
        
- **Sección de Credenciales:**
    
    - Campo de entrada para el **Correo electrónico**.
        
    - Fila agrupada: Campo de entrada para **Contraseña** y Campo de entrada para **Confirmar contraseña**.
        
- **Sección de Acción Principal:**
    
    - Botón principal de llamado a la acción: _"Crear Cuenta"_.
        
- **Pie del Contenedor:**
    
    - Texto informativo con enlace para redirigir: _"¿Ya tienes cuenta? **Iniciar sesión**_" (apunta a `/inicio`).
