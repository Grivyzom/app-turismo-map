### Vista: `/inicio` (Inicio de Sesión)

**1. Contenedor Principal (La Isla Central)**

- **Encabezado:** Saludo de bienvenida y logotipo o nombre de la aplicación.
    
- **Sección de Formulario:**
    
    - Campo de entrada para el **Correo electrónico**.
        
    - Campo de entrada para la **Contraseña** (con opción visual para ocultar/mostrar texto).
        
- **Sección de Recuperación:**
    
    - Enlace alineado de forma discreta: _"Olvidé mi contraseña"_.
        
- **Sección de Acción Principal:**
    
    - Botón principal de llamado a la acción: _"Iniciar Sesión"_ o _"Ingresar"_.
        
- **Divisor Visual:** * Línea o espacio con el texto _"o"_.
    
- **Sección de Acceso Rápido (Social):**
    
    - Botón destacado para inicio de sesión con **Google**.
        
- **Pie del Contenedor:**
    
    - Texto informativo con enlace para redirigir: _"¿No tienes cuenta? **Registrarse**_" (apunta a `/registro`).
